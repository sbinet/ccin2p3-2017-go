# ccin2p3-2017-go

`ccin2p3-2017-go` is a repository to test refactoring, formating, code inspection, ... in Go and for Go.

## Dependencies

```sh
$> go get -u -v \
	github.com/sbinet/ccin2p3-2017-go/... \
	golang.org/x/tools/cmd/{eg,gorename,guru} \
	github.com/golang/lint/golint
```
